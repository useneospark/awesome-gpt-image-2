# Contributing to Awesome GPT Image 2 Prompts

Thank you for your interest in contributing! This repository is a curated collection of GPT Image 2 prompts, maintained by the community and powered by [NeoSpark](https://useneospark.com).

## How to Contribute

### Adding a New Prompt

1. **Fork** this repository
2. **Create a branch** for your contribution: `git checkout -b add-prompt-name`
3. **Add your prompt** to the appropriate category in the README or create a new file in `/prompts/`
4. **Follow the format** (see below)
5. **Submit a Pull Request** with a clear description

### Prompt Format

Each prompt should include:

```markdown
### Prompt Title

> Full prompt text here. Be specific and detailed.
> Include camera details, lighting, style references, and mood.

**Category:** Category Name
**Use Case:** Brief description of ideal use
**Model Tested:** GPT Image 2

[**Try on NeoSpark →**](https://useneospark.com/prompt-lib?prompt=your-slug&ref=awesome-gpt-image-2)
```

### Quality Guidelines

- **Tested**: Prompt must produce good results with GPT Image 2
- **Specific**: Include technical details (lens, lighting, style references)
- **Original**: Don't copy prompts verbatim from other sources without attribution
- **Text accuracy**: Use quotation marks for any text elements
- **One prompt per PR**: Unless it's a themed set/series

### Categories

If your prompt doesn't fit existing categories, suggest a new one in your PR description.

Current categories:
- Cinematic & Film
- Portraits & Beauty
- Product Photography
- UI & App Design
- Fantasy & Anime
- Nature & Landscapes
- Marketing & Social Media
- Typography & Posters
- Architecture & Interior
- Experimental & Fun

### Code of Conduct

- Be respectful and constructive in discussions
- Credit original creators when adapting existing prompts
- Focus on quality over quantity

## Reporting Issues

Found a broken link or incorrect prompt? Open an issue with:
- Description of the problem
- Location (file/line)
- Suggested fix

## Questions?

Join the discussion or reach out via [NeoSpark's community](https://useneospark.com).
