## GPT Image 2란?

**GPT Image 2**는 OpenAI의 최신 이미지 생성 모델로, ChatGPT의 네이티브 이미지 생성 기능을 제공합니다.

- **사진급 실제감** — 정확한 해부학, 조명, 텍스처
- **완벽한 텍스트 렌더링** — 인용문, 표지판, UI 라벨이 선명하고 읽기 쉬움
- **스타일 일관성** — 대화 내 여러 이미지에서 통일된 스타일
- **고급 편집** — 자연어로 기존 이미지 수정
- **멀티모달 이해** — 이전 메시지의 맥락 이해

## 프롬프트 엔지니어링 팁

### 8슬롯 공식

1. **주제** — 메인 포커스는 무엇인가?
2. **액션/포즈** — 무엇을 하고 있는가?
3. **환경** — 어디에 있는가?
4. **조명** — 어떻게 비춰지는가?
5. **카메라/렌즈** — 어떤 장비를 사용?
6. **스타일/참조** — 어떤 미학?
7. **컬러 그레이드** — 어떤 팔레트?
8. **분위기** — 어떤 느낌?

### 텍스트 정확도 향상

- 텍스트를 **따옴표**로 감싸기: `"OPEN LATE"`
- **글꼴 스타일** 지정: "bold sans-serif"
- **배치** 설명: "top center"
- **강조는 전체 대문자**: `"DREAM BIG"`
