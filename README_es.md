## Consejos de Ingeniería de Prompts

### La Fórmula de 8 Elementos

1. **Sujeto** — ¿Cuál es el foco principal?
2. **Acción/Pose** — ¿Qué están haciendo?
3. **Entorno** — ¿Dónde están?
4. **Iluminación** — ¿Cómo está iluminado?
5. **Cámara/Lente** — ¿Qué equipo?
6. **Estilo/Referencia** — ¿Qué estética?
7. **Gradación de Color** — ¿Qué paleta?
8. **Ambiente** — ¿Qué sensación?

### Precisión de Texto

- Poner texto entre **comillas**: `"OPEN LATE"`
- Especificar **estilo de fuente**: "bold sans-serif"
- Describir **ubicación**: "top center"
- Usar **MAYÚSCULAS** para énfasis: `"DREAM BIG"`
