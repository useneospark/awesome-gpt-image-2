### What is GPT Image 2?

GPT Image 2 is OpenAI's latest image generation model, available natively in ChatGPT. It produces photorealistic images with excellent text rendering and style consistency.

### How is this different from DALL-E 3?

GPT Image 2 offers improved photorealism, better text accuracy, more consistent style across multiple images, and advanced editing capabilities. It's the successor to DALL-E 3.

## Using Prompts

### Why aren't my results matching the examples?

Results vary based on:
- The exact model version you're using
- Random seed variations
- Specific phrasing and punctuation
- Model updates over time

Try small variations in wording or add more specific technical details.

### How do I get better text in my images?

GPT Image 2 handles text best when you:
1. Put text in **quotation marks**: `"YOUR TEXT HERE"`
2. Specify font style: "bold sans-serif" or "elegant script"
3. Describe placement: "centered at top" or "bottom right corner"
4. Use ALL CAPS for emphasis

### Can I modify existing images?

Yes! GPT Image 2 supports image editing. Upload an image and describe the changes you want.
