## Conseils d'Ingénierie de Prompts

### La Formule des 8 Éléments

1. **Sujet** — Quel est le focus principal ?
2. **Action/Pose** — Que font-ils ?
3. **Environnement** — Où sont-ils ?
4. **Éclairage** — Comment est-ce éclairé ?
5. **Caméra/Objectif** — Quel équipement ?
6. **Style/Référence** — Quelle esthétique ?
7. **Gradation de Couleur** — Quelle palette ?
8. **Ambiance** — Quelle sensation ?

### Précision du Texte

- Mettre le texte entre **guillemets** : `"OPEN LATE"`
- Spécifier le **style de police** : "bold sans-serif"
- Décrire le **placement** : "top center"
- Utiliser les **MAJUSCULES** pour l'emphase : `"DREAM BIG"`
