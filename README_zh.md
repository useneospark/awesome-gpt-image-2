## 提示工程技巧

基于 GPT Image 2 的大量测试，以下是产生最佳结果的模式：

### 八要素公式

按顺序用这些要素构建你的提示词：

1. **主体** — 主要焦点是什么？（人物、产品、建筑）
2. **动作/姿势** — 他们在做什么？（站立、漂浮、奔跑）
3. **环境** — 他们在哪里？（东京街头、工作室、森林）
4. **灯光** — 如何照明？（黄金时段、伦勃朗、霓虹）
5. **相机/镜头** — 什么设备？（85mm f/1.4、24mm 广角）
6. **风格/参考** — 什么美学？（韦斯·安德森、Vogue、皮克斯）
7. **色调** — 什么调色板？（青橙、粉彩、单色）
8. **氛围/气氛** — 什么感觉？（忧郁、史诗、舒适）

### 文字准确性

GPT Image 2 在以下情况下处理文字效果极佳：
- 将文字放在**引号中**：`"OPEN LATE"` 或 `'SALE 50% OFF'`
- 指定**字体风格**："粗体无衬线" 或 "优雅草书"
- 描述**位置**："顶部居中" 或 "右下角"
- 使用**全大写强调**：`"DREAM BIG"` 比大小写混合渲染更可靠

### 风格锚点

不要使用模糊描述，使用具体参考：

| 不要写 | 要写 |
|--------|------|
| "美丽的奇幻艺术" | "Ufotable 动画工作室制作质量" |
| "专业照片" | "哈苏 H6D 拍摄，100mm 镜头 f/2.8" |
| "卡通风格" | "皮克斯 3D 渲染，电影级质量" |
| "老照片质感" | "Kodak Portra 400，35mm 胶片颗粒，漏光" |
| "暗黑忧郁" | "明暗对比灯光，深邃阴影，伦勃朗风格" |

### 品牌一致性

对于系列作品，在每个提示词后附加**项目风格块**：

```
项目风格块：
- 品牌调色板：深海蓝 #0f172a、电光青 #38bdf8、暖奶油 #fef3c7
- 字体：几何无衬线标题，粗衬线正文
- 氛围：干净、自信、略带未来感，从不幼稚
- 约束：背景中不要随机人物，不要无标题 UI 元素
```
