**GPT Image 2** is OpenAI's latest image generation model, powering the native image creation capabilities in ChatGPT. It delivers:

- **Perfect text rendering** — quotes, signs, UI labels are crisp and legible
- **Style consistency** across multiple images in a conversation
- **Advanced editing** — modify existing images with natural language
