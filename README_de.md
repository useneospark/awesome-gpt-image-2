## Prompt Engineering Tipps

### Die 8-Slot-Formel

1. **Subjekt** — Was ist der Hauptfokus?
2. **Aktion/Pose** — Was machen sie?
3. **Umgebung** — Wo sind sie?
4. **Beleuchtung** — Wie ist es beleuchtet?
5. **Kamera/Objektiv** — Welche Ausrüstung?
6. **Stil/Referenz** — Welche Ästhetik?
7. **Farbgebung** — Welche Palette?
8. **Stimmung** — Welches Gefühl?

### Textgenauigkeit

- Text in **Anführungszeichen** setzen: `"OPEN LATE"`
- **Schriftstil** angeben: "bold sans-serif"
- **Platzierung** beschreiben: "top center"
- **GROSSBUCHSTABEN** für Betonung: `"DREAM BIG"`
